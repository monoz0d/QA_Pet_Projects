# Тест-кейсы для проекта "Генератор паролей"

## NG-001: Генерация одного пароля с цифрами
**Приоритет:** Высокий  
**Предусловия:** Программа запущена  
**Шаги выполнения:**  
1. Ввести количество паролей: 1  
2. Ввести длину пароля: 8  
3. Включить цифры: да  
4. Включить буквы и символы: нет  
5. Исключить неоднозначные символы: нет  
**Ожидаемый результат:** Пароль длиной 8 символов, состоящий только из цифр
