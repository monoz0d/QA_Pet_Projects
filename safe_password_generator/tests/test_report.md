# Отчет по результатам проведения тестов — Генератор паролей

## Краткое описание
В ходе тестирования был применен метод 12 тест-кейсов для проверки работы программы "Генератор паролей".  

Суть тестирования заключается в проверке корректной генерации паролей с разными наборами символов, граничных случаев и обработки некорректного ввода.

---

## Сведения о ходе проверок
- Всего тестов: 12  
- Пройдено: 9  
- Провалено: 3
