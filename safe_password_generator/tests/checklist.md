# Чеклист тестирования проекта "Генератор паролей"

### Общие проверки
- [ ] Программа запускается без ошибок  
- [ ] Возможен ввод количества паролей и длины одного пароля  
- [ ] Возможен выбор включения цифр, строчных и прописных букв, спецсимволов  
- [ ] Возможен выбор исключения неоднозначных символов (`il1Lo0O`)  
- [ ] Корректная обработка некорректного ввода (не `да`/`нет`)  

### Позитивные сценарии
- [ ] Генерация одного пароля с цифрами  
- [ ] Генерация одного пароля со строчными буквами  
- [ ] Генерация одного пароля с прописными буквами  
- [ ] Генерация одного пароля с символами `!#$%&*+-=?@^_`  
- [ ] Генерация нескольких паролей заданной длины  

### Краевые случаи
- [ ] Длина пароля = 1  
- [ ] Длина пароля = максимально допустимая (например, 100)  
- [ ] Количество паролей = 1  
- [ ] Количество паролей = большое число (например, 1000)  
- [ ] Исключение неоднозначных символов из всех групп  

### Негативные сценарии
- [ ] Ввод некорректного ответа вместо `да`/`нет`  
- [ ] Пустой ввод при выборе включения символов  
- [ ] Невозможность генерации пароля при пустом наборе символов  

### Дополнительные проверки
- [ ] Проверка, что сгенерированные пароли имеют указанную длину  
- [ ] Проверка, что пароли включают только выбранные символы  
- [ ] Проверка стабильности программы при повторной генерации
