# Генератор паролей

Мини-проект: генератор случайных паролей на Python с возможностью настройки состава символов.

## Функциональность
- Генерация одного или нескольких паролей указанной длины
- Настройка включения/исключения:
  - Цифр (`0-9`)
  - Строчных латинских букв (`a-z`)
  - Прописных латинских букв (`A-Z`)
  - Специальных символов (`!#$%&*+-=?@^_`)
  - Неоднозначных символов (`il1Lo0O`)  
- Проверка корректности пользовательского ввода (только `да` или `нет`)
- Поддержка повторного запроса при некорректном вводе  

## Как использовать
1. Запустить скрипт `password_generator.py`
2. Следовать подсказкам:
   * Ввести количество паролей для генерации
   * Ввести длину одного пароля
   * Ответить на вопросы о включении цифр, букв и символов
   * Опционально исключить неоднозначные символы
3. Программа выведет сгенерированные пароли в консоль
