# Числовая угадайка

Мини-проект: консольная игра на Python, в которой нужно угадать случайное число в заданном диапазоне.

## Функциональность
- Генерация случайного числа в диапазоне от 1 до заданной границы  
- Ввод пользователем чисел с проверкой корректности  
- Подсказки: больше или меньше загаданного числа  
- Подсчет количества попыток  
- Возможность сыграть несколько раз подряд  
- Обработка некорректного ввода (число вне диапазона)

## Как играть
1. Запустить скрипт  
2. Ввести правую границу диапазона (например, 100)  
3. Пытаться угадать число:
   - Вводите числа от 1 до указанной границы  
   - Программа подскажет, больше или меньше ваше число  
4. Игра закончится, когда число угадано  
5. Можно начать новую игру по желанию
