# Баг репорты — Числовая угадайка

## BUG-001  
### Ввод числа: падение программы при вводе букв (NG-006)

### Окружение
- Dev / Staging  
- Python 3.14  
- Консольное приложение  

### Связанные артефакты
- Тест-кейс: NG-006  
- Чек-лист: негативные сценарии (ввод букв вместо числа)  
- Тип тестирования: негативное  

### Шаги воспроизведения
1. Запустить программу  
2. Ввести границу диапазона (например, `10`)  
3. Ввести значение `да` (или любые буквы)  
4. Нажать Enter  

### Ожидаемый результат
- Программа не падает  
- Выводится сообщение о некорректном вводе  
- Пользователю предлагается повторить ввод  

### Фактический результат
- Программа завершается с ошибкой:  
```ValueError: invalid literal for int() with base 10: 'да'```

### Severity
Высокий  

### Priority
Высокий  

### Обоснование
- **Severity:** происходит падение программы — критическая ошибка  
- **Priority:** баг легко воспроизводится и связан с пользовательским вводом (частый сценарий), требует немедленного исправления
