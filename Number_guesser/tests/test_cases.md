# Тест-кейсы для проекта "Числовая угадайка"

## NG-001: Угадывание числа с первой попытки
**Приоритет:** Высокий  
**Предусловия:** Программа запущена  

**Шаги выполнения:**  
1. Ввести границу диапазона `3`  
2. Ввести загаданное число (угадать)  

**Ожидаемый результат:**  
- Вывод сообщения о победе  
- Количество попыток = 1
