# Чеклист тестирования проекта "Числовая угадайка"

### Общие проверки
- Программа запускается без ошибок  
- Запрашивается граница диапазона  
- Генерируется случайное число  

### Позитивные сценарии
- Угадывание числа с первой попытки  
- Угадывание числа за несколько попыток  
- Корректные подсказки (больше/меньше)  
- Подсчет попыток работает корректно  
- Возможность начать новую игру  

### Краевые случаи
- Граница = 1  
- Очень большая граница (>1000)  
- Минимальное количество попыток (1)  

### Негативные сценарии
- Ввод букв вместо числа  
- Ввод числа вне диапазона  
- Пустой ввод  
- Ввод отрицательных чисел  

### Дополнительные проверки
- Повтор игры работает корректно  
- Регистр ответа "да/нет" (Да, ДА, yes — если расширять)  
- Быстрые повторные вводы
