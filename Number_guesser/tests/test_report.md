# Отчет по результатам проведения тестов

## Краткое описание
В ходе тестирования был применен метод 10 тест-кейсов для проверки работы программы "Числовая угадайка"

Суть тестирования заключается в проверке основной игровой логики и обработки пользовательского ввода
