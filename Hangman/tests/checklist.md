## Чеклист тестирования проекта "Виселица"

### Общие проверки
- Программа запускается без ошибок
- Загаданное слово выбирается случайно
- Возможен ввод буквы или полного слова
- Программа отслеживает количество попыток (6) и отображает виселицу

### Позитивные сценарии
- Угадывание буквы, которая есть в слове
- Угадывание буквы, которой нет в слове (уменьшение попыток)
- Угадывание полного слова правильно (победа)
- Угадывание полного слова неверно (уменьшение попыток)
- Повторный ввод буквы или слова корректно обрабатывается

### Краевые случаи
- Последняя (6-я) неверная попытка → игра заканчивается проигрышем
- Слово с повторяющимися буквами → все вхождения открываются при угадывании буквы
- Загаданное слово полностью угадано до окончания попыток → победа

### Негативные сценарии
- Ввод цифр → вывод сообщения "Введите только буквы!"
- Ввод спецсимволов → вывод сообщения "Введите только буквы!"
- Ввод пустой строки → вывод сообщения "Введите только буквы!"
- Повторный ввод уже угаданной буквы → сообщение "Эта буква уже была!"

### Дополнительные проверки
- Сохранение регистра букв (строчные и заглавные буквы корректно отображаются)
- Смешанный ввод букв и слова
- Игровой процесс корректно отображает прогресс слова и виселицу после каждого ввода
