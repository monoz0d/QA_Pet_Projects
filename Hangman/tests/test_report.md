# Отчет по результатам проведения тестов проекта "Виселица"

## Краткое описание
В ходе тестирования было выполнено 10 тест-кейсов для проверки работы программы "Виселица".  

Суть тестирования заключается в проверке всех ключевых функций игры: угадывания букв и слов, корректного отображения прогресса, виселицы и обработки некорректного ввода.
