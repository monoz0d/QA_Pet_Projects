# Виселица 

Мини-проект: игра "Виселица" на Python с поддержкой русского языка.

## Функционал
- Угадывание слов из заданного списка
- Отображение прогресса слова с буквами и подчеркиваниями
- Игровой процесс с количеством попыток (6) и визуальным отображением виселицы
- Проверка ввода: буквы и слова
- Сохранение введенных букв и слов
- Обработка ошибок при повторном вводе

## Как играть
1. Запустить скрипт  hangman.py
2. Следовать подсказкам:
* Вводите букву или слово
* Программа покажет текущее состояние слова
* Попытки уменьшаются при неверных догадках
3. Игра заканчивается победой (угадали слово) или проигрышем (закончились попытки)
## Пример игры
```text
Давайте играть!
Слово: ____ (4 букв)
Введи букву или слово:

<а>

Есть такая буква!
Слово: _А__ 
Буквы: ['А']
```
