# Чеклист тестирования проекта "Калькулятор систем счисления"

### Общие проверки
- [ ] Программа запускается без ошибок  
- [ ] Возможен выбор режима: перевод **в десятичную** или **из десятичной** системы  
- [ ] Возможен ввод основания системы счисления от 2 до 16  
- [ ] Обработка букв A–F корректна для 16-ричной системы  

### Позитивные сценарии
- [ ] Перевод числа из двоичной системы в десятичную  
- [ ] Перевод числа из восьмеричной системы в десятичную  
- [ ] Перевод числа из десятичной системы в двоичную  
- [ ] Перевод числа из десятичной системы в шестнадцатеричную  

### Краевые случаи
- [ ] Число = 0  
- [ ] Основание = 2  
- [ ] Основание = 16  
- [ ] Очень большое число (>100000)  

### Негативные сценарии
- [ ] Ввод букв вместо числа для основания < 11  
- [ ] Ввод некорректного символа для выбранного основания  
- [ ] Пустой ввод числа  
- [ ] Основание вне диапазона (например, 1 или 17)  

### Дополнительные проверки
- [ ] Смешанный ввод (буквы + цифры) для 16-ричной системы  
- [ ] Проверка регистра букв (должны корректно интерпретироваться как A–F)  
- [ ] Проверка на отрицательные числа (должны быть корректно обработаны или отклонены)
