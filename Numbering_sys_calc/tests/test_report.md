# Отчет по результатам проведения тестов - Калькулятор систем счисления

## Краткое описание
В ходе тестирования был применен метод 9 тест-кейсов для проверки работы программы "Калькулятор систем счисления".  

Суть тестирования заключается в проверке корректного перевода чисел между системами счисления и обработки некорректного ввода.

---

## Сведения о ходе проверок
- Всего тестов: 9  
- Пройдено: 6  
- Провалено: 3
