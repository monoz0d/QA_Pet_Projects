# Тест-кейсы для проекта "Калькулятор систем счисления"

## NSC-001: Перевод числа из двоичной системы в десятичную
**Приоритет:** Высокий  
**Предусловия:** Программа запущена, выбран режим перевода в десятичную  
**Шаги выполнения:**  
1. Ввести число: 1011  
2. Ввести систему счисления: 2  
3. Нажать Enter  
**Ожидаемый результат:** 11
