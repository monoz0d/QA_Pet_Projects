# Калькулятор систем счисления

Мини-проект: программа для перевода чисел между различными системами счисления (от 2 до 16).

## Функционал
- Перевод числа **в десятичную систему**
- Перевод числа **из десятичной системы**
- Поддержка систем счисления от **2 до 16**
- Работа с цифрами и буквами (A–F для 16-ричной системы)
- Проверка корректности ввода системы счисления

## Пример использования

```text
Переведем в десятичную систему или из десятичной? (1 - в дес., 2 - из дес.): 

<1>

Введите число: 

<1A>

Введите систему счисления (2–16):

<16>

Результат: 26
