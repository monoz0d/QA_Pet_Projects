### Тут собраны все мои пет-проекты. 
Цель репозитория помимо практики - демонстрация навыков тестирования и немного программирования.
---

## 📂 Структура репозитория
Каждый проект оформлен как отдельная папка с собственным `README.md` и включает три основные подпапки:

| Папка           | Содержание |
|-----------------|------------|
| `src/`         | Код проекта (`*.py`) |
| `bug reports_/`         | • Баг-репорты (`bug_reports.md`) <br>• Папка `attachments/` с вложениями |
| `tests/`         | Тестовая документация: <br>• `requirements.md`<br>• `test_plan.md` <br>• `test_cases.md` <br>• `checklist.md` <br>• `test_report.md` |
