# Тест-кейсы для проекта Caesar Cipher

## CC-001: Шифрование текста с одним шагом сдвига (английский алфавит)
**Приоритет:** Высокий  
**Предусловия:**  
1. Программа запущена.  
2. Выбран режим шифрования.  
3. Выбран английский алфавит.  

**Шаги выполнения:**  
1. Ввести текст `abc`.  
2. Ввести шаг сдвига `1`.  
3. Нажать Enter.  

**Ожидаемый результат:**  
- Результат: `bcd`.  
- Регистр букв сохраняется.  
- Цифры и спецсимволы остаются без изменений.

## CC-006: Шаг сдвига = длина алфавита
**Приоритет:** Средний  
**Предусловия:** Программа запущена.  
**Шаги выполнения:**  
1. Ввести текст `abc`.  
2. Ввести шаг сдвига, равный длине алфавита.

**Ожидаемый результат:**  
- Результат совпадает с исходным текстом (`abc`).
