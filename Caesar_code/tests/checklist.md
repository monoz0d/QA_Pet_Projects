## Чеклист тестирования проекта Caesar Cipher

### Общие проверки
- Программа запускается без ошибок
- Возможен выбор режима: шифрование / дешифрование
- Возможен выбор языка: русский / английский
- Поддержка одного шага сдвига и диапазона

### Позитивные сценарии
- Шифрование английского текста с шагом 1  
- Шифрование русского текста с шагом 1  
- Дешифрование английского текста с шагом 1  
- Дешифрование русского текста с шагом 1  
- Шифрование с диапазоном сдвига (проверка всех вариантов)

### Краевые случаи
- Пустой текст: программа не падает  
- Шаг сдвига = 0: текст остаётся без изменений  
- Шаг сдвига = длина алфавита: текст остаётся без изменений  
- Очень длинный текст (>1000 символов): результат корректен  

### Негативные сценарии
- Ввод букв вместо числа: программа запрашивает повторный ввод  
- Потеря регистра при дешифровании: буквы должны сохранять регистр  
- Ввод текста со спецсимволами: спецсимволы остаются без изменений, не ломают вывод
