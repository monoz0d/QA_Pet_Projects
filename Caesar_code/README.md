# Шифр Цезаря

Мини-проект: шифр Цезаря с поддержкой русского и английского алфавита и диапазона сдвигов.  

## Функциональность
- Шифрование и дешифрование текста  
- Выбор языка: русский / английский  
- Один шаг сдвига или диапазон  
- Сохранение регистра букв, игнорирование цифр и символов  

## Пример использования
```text
Введите текст:

<Hello>

Шифрование или дешифрование? 1 - шифр., 2 - дешифр.:

<1>

Английский или русский? 1 - англ., 2 - рус.:

<1>

Укажите шаг сдвига:

<3>

Результат: Khoor
