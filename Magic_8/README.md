# Магический шар
Мини-проект: программа, которая отвечает на любой заданный вопрос случайным образом.

## Функционал
- Поздороваться с пользователем
- Спрашивать имя пользователя
- Отвечать на любой вопрос из заранее подготовленного списка ответов
- Возможность задавать несколько вопросов подряд
- Завершение программы по желанию пользователя

## Пример использования

```text
Привет Мир, я магический шар, и я знаю ответ на любой твой вопрос
Как тебя зовут? 

<Никита> 

Привет, Никита
Задай свой вопрос! 

<Будет ли завтра солнечно?>

Определенно да
Хочешь задать еще вопрос? 

<да>

Задай свой вопрос! 

<Стоит ли проводить тестирование?>

Вероятнее всего
Хочешь задать еще вопрос? 

<нет>

Возвращайся, если возникнут вопросы!
```
