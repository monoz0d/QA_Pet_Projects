# Баг репорты — Магический шар 


## BUG-001  
### Приветствие: некорректный вывод при пустом имени (MB-004)

### Окружение
- Dev / Staging  
- Python 3.14  
- Консольное приложение  

### Связанные артефакты
- Тест-кейс: MB-004  
- Чек-лист: краевые случаи (пустое имя пользователя)  
- Тип тестирования: негативный  

### Шаги воспроизведения
1. Запустить программу  
2. На запрос имени нажать Enter (ввести пустую строку)  

### Ожидаемый результат
- Программа выводит нейтральное приветствие (например: `Привет!`) 

### Фактический результат
- Программа выводит: `Привет, ` (пустое имя)  

### Severity
Низкий  

### Priority
Средний  

### Обоснование
- **Severity:** баг не влияет на основной функционал (программа продолжает работать)  
- **Priority:** влияет на пользовательский опыт, желательно исправить перед релизом

## BUG-002  
### Логика продолжения: завершение программы при пустом вводе (MB-008)

### Окружение
- Dev / Staging  
- Python 3.14  
- Консольное приложение  

### Связанные артефакты
- Тест-кейс: MB-008  
- Чек-лист: негативные сценарии (пустой ввод при подтверждении)  
- Тип тестирования: негативный  

### Шаги воспроизведения
1. Запустить программу  
2. Ввести имя  
3. Задать любой вопрос  
4. На вопрос "Хочешь задать еще вопрос?" нажать Enter (пустой ввод)  

### Ожидаемый результат
- Программа повторно запрашивает ввод (`да`/`нет`)  
- Программа не завершается  

### Фактический результат
- Программа завершает работу  

### Severity
Средний  

### Priority
Высокий  

### Обоснование
- **Severity:** нарушается логика работы программы (некорректная обработка ввода)  
- **Priority:** пользователь может случайно завершить программу → критично для UX

### Окружение

- Python 3.14

2. Ввести имя

- Программа не завершается  

### Фактический результат

### Severity
Средний  

### Priority
Высокий

- **Priority:** высокая вероятность ошибки пользователя -> ухудшение UX и преждевременное завершение программы
