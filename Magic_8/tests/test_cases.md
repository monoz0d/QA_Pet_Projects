## Тест-кейсы для проекта Магический шар

### MB-001: приветствие пользователя

**Приоритет:** высокий

**Предусловия:** 
- Программа запущена.

**Шаги выполнения:**
1. Ввести имя пользователя `Никита`.
2. Нажать Enter.

**Ожидаемый результат:**
* Вывод: `Привет, Никита`
* Программа готова к приёму вопросов

---

### MB-002: ответ на один вопрос

**Приоритет:** высокий

**Предусловия:** 
- Программа запущена, пользователь ввёл имя.

**Шаги выполнения:**
1. Ввести вопрос: `Будет ли завтра солнечно?`
2. Нажать Enter.

**Ожидаемый результат:**
* Появляется один из заранее заданных ответов (`Бесспорно`, `Спроси позже`, и т.д.)
* Программа не падает

---

### MB-003: несколько вопросов подряд

**Приоритет:** высокий

**Предусловия:** 
- Программа запущена, пользователь ввёл имя.

**Шаги выполнения:**

1. Задать первый вопрос `Стоит ли проводить тестирование?`
2. После ответа ввести `да` для продолжения
3. Задать второй вопрос `Получится ли сдать экзамен?`
4. Ввести `нет` для завершения

**Ожидаемый результат:**

* На каждый вопрос выводится ответ из списка
* Программа корректно завершает работу после ввода `нет`

---
