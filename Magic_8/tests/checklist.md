## Чеклист тестирования проекта Магический шар

### Общие проверки

- Программа запускается без ошибок
- Выводится приветствие и запрос имени пользователя
- Пользователь может задать один или несколько вопросов
- Программа завершает работу по желанию пользователя

### Позитивные сценарии

- Пользователь вводит корректное имя и получает приветствие
- Пользователь задаёт вопрос и получает ответ из списка
- Пользователь задаёт несколько вопросов подряд
- Программа корректно завершает работу при отказе от дальнейших вопросов

### Краевые случаи

- Пустое имя пользователя
- Пустой вопрос
- Вопрос с очень длинной строкой (>1000 символов)
- Вопрос, содержащий спецсимволы или emoji

### Негативные сценарии

- Пользователь вводит пустой ответ на запрос продолжения (`да`/`нет`)
- Ввод текста вместо ожидаемого `да`/`нет` при подтверждении продолжения
- Вопрос содержит только пробелы или невидимые символы

### Дополнительные проверки

- Вопрос на разных языках (русский, английский, смешанный)
- Проверка всех вариантов ответов из списка (рандомизировано)
- Unicode символы в вопросе (emoji, спецсимволы)
